# Neutral TS Starter Py

**A modular Python PWA scaffold built on Neutral TS. Extend with reusable components.**

## What is this?

Neutral TS Starter Py is an opinionated starter template that combines:

- **Python backend** Flask
- **Neutral TS** for universal templating
- **PWA-ready** architecture
- **Pre-configured build system** for development & production
